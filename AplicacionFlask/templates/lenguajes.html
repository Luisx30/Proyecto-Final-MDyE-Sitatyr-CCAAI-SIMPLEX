Mas información sobre el método SIMPLEX

El algoritmo simplex es un método algebraico con fundamentos geométricos que nos ayuda a encontrar soluciones de problemas de programación lineal, que fue realizado por George Dantzig en 1947. Para poder realizar este método es importante transformar el procedimiento geométrico en uno algebraico que se basa en solucionar sistemas de ecuaciones, es decir, llevarlo a su forma estándar.

El algoritmo simplex de dos fases, es una variante del algoritmo simplex que permite resolver aquellos problemas de programación lineal que luego de llevarlos a su forma estándar, no se puede llegar a una solución factible y para esto se utilizan variables artificiales.

Esta página web permite la resolución de matrices mediante el algoritmo simplex y despliegue las iteraciones realizadas para que el usuario pueda observar cómo se resuelve el problema mediante este método.
